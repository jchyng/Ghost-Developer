# Ghost Developer - 향후 확장 계획 (v3.0+)

> v2.0(Orchestrator + Chat UI)이 안정적으로 동작한 이후, 실제 사용하며 필요성이 느껴질 때 추가할 기능들.

---

## 1. Telegram 원격 알림 & 개입

**배경:** 자리를 비운 상태에서 Orchestrator가 막히거나 사용자 판단이 필요할 때.

- 주요 상태 변화(작업 완료, 에러, rate limit 대기 중) 시 Telegram 알림
- Telegram에서 채팅 메시지 전송 → Orchestrator에게 지시 가능
- Inline Keyboard로 간단한 선택지 제공 (계속/중단/다시시도)

---

## 2. Circuit Breaker

**배경:** Orchestrator가 같은 에러로 반복 루프를 돌며 비용만 낭비할 때.

- 동일 에러 패턴 N회 이상 감지 시 자동 일시정지
- 사용자 개입 요청 (채팅 알림 + Telegram)
- 재개 / 다른 접근 지시 후 재시작

---

## 3. 멀티 유저 & 인증

**배경:** 팀원과 공유하거나 외부에서 접근할 때.

- JWT 기반 인증
- 사용자별 Chat 세션 격리
- 공유 세션 (read-only observer 모드)

---

## 4. 비용 & 사용량 대시보드

**배경:** Orchestrator + Claude Code 호출 비용이 쌓일 때.

- 세션별 token usage, 예상 비용 트래킹
- 일별/주별 사용량 그래프
- 예산 초과 시 자동 일시정지 옵션
